# MacroLens

A macroeconomic market analysis dashboard that displays real-time stock data and AI-powered insights.

## What it does
- Fetches live stock prices for any publicly traded ticker
- Displays a month of historical price data as an interactive chart
- Shows macro indicators, market sentiment, and evidence-based analysis
- Built to democratise the kind of market insight usually reserved for Wall Street

## Tech stack
- Frontend: React + Chart.js
- Backend: Python + FastAPI + yfinance
- Deployment: Vercel + Render (coming soon)

## Status
Active development — currently building the AI reasoning layer.
